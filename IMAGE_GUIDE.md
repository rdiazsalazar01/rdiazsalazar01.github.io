## DEPLOYING TO GITHUB PAGES
1. Create a new GitHub repository (e.g. `rodrigodiaz.github.io` or `portfolio`)
2. Upload the entire `portfolio/` folder contents to the repository root
3. Go to Settings → Pages → Source: Deploy from branch → main → / (root)
4. Your site will be live at `https://yourusername.github.io` in a few minutes
